## Usage

This plugin is only compatible with the `main` branch of Bevy (post-`0.5` version, with new renderer).

### System setup

Add the Hanabi plugin to your app:

```rust
App::default()
    .add_default_plugins()
    .add_plugin(HanabiPlugin)
    .run();
```
